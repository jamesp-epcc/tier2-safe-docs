Cirrus SAFE Documentation¶

This is the documentation for the SAFE service management web interface provided by EPCC for the Cirrus facility. The Cirrus SAFE service provides a web-based interface for managing access and resources on Cirrus.

To log into Cirrus SAFE please go to:

  • Cirrus SAFE Login

Cirrus SAFE Overview¶

The SAFE is a web-based application that is used for:

  • Administering user accounts and projects on Cirrus
  • Reporting on usage of the Cirrus service

Cirrus SAFE for individual users¶

All users on Cirrus have accounts on the Cirrus SAFE through which they can view the resources they have access to, administer their login accounts on Cirrus and query their usage of the various Cirrus resources.

  • SAFE for Individual Users
    • SAFE: Registering, logging in, passwords
    • Cirrus Facility: Accounts, passwords
    • User Mailing Options
    • Tracking Resource Usage

Cirrus SAFE for project leaders¶

Project leaders (PI’s and any designated project managers) can also use the SAFE to approve requests to join the project from individual users and generate reports on the project usage on the Cirrus facilities the project has access to.

  • SAFE for Project Leaders
    • Getting Started
    • Viewing your allocated resources
    • Managing Project Users
    • Tracking your Project Usage
